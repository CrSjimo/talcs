Detailed Description

The class takes an PositionableAudioSource object as the result of a asynchronous process.

The PositionableAudioSource is the result of a asynchronous process (i.e. synthesizing, loading from the Internet) and will be ready at a later point of time. QFuture is used for the implementation of this class. (But the Qt Concurrent framework is not used, only QFuture and other relevant classes are used)

This object requires some restrictions for the QFuture object:

  • The QFuture::progressMinimum() should be set to 0 and the QFuture::progressMaximum() should be set to the length of the source.
  • It should be started by QFutureInterface::reportStarted() before used by this object.
  • There should be only one result in it. Before finished, the QFuture::progressValue() should be set to the maximum and the result should be reported.
See also
QFuture, QFutureWatcher, QFutureInterface
