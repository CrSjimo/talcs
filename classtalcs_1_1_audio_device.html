◆ lock()

void lock ( )
virtual

Locks the audio callback.

Only lightweight work should be done while the audio device is locked. If any heavy work is going to be done, stop the audio device first, do the work, and start it again.

The audio device must be unlocked when it is going to stop or close, otherwise these functions will be blocked.

Reimplemented in ASIOAudioDevice, and SDLAudioDevice.
