AudioDevice * createDevice ( const QString &  name)

Creates an audio device.

The parent of the device created is set to this object, and use delete to deallocate.

nullptr will be returned if the function fails to create the device with the specified name. Note that the audio device is not guaranteed to work, and use to check whether it can work or not. AudioDevice, AudioDevice::isInitialized()
